# Feedback-Icons

`feedback-icon-sheet.png` ist ein transparentes 3×2-Sheet mit sechs Frames zu
je 512×512 Pixeln: Rakete, Granate, Geländebrecher, Teamgesundheit,
Zugreihenfolge und Managerkommando.

Alle sechs Icons werden in Match-HUD oder Einsatzplanung verwendet. Sie sind
visuelle Orientierung und besitzen keine Autorität über Spielregeln.

Generierung und Aufbereitung sind in `docs/ASSET_GENERATION.md` dokumentiert.
