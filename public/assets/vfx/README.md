# Comic-VFX

`comic-vfx-sheet.png` ist ein generierter 2×2-Atlas für Raketenabgas,
Einschlag, Rauch und Terraintrümmer.

`secondary-vfx-sheet.png` ergänzt zwei kurze, jeweils zweiphasige Effekte:
Granatenabpraller in den Frames 0–1 und Landestaub in den Frames 2–3. Beide
werden nur von bereits feststehenden Simulationsereignissen ausgelöst und
verändern weder Position noch Kollision.

Die vollständige Generierungsanweisung steht in `docs/ASSET_GENERATION.md`.
