# VFX-Platzhalter

`comic-vfx-sheet.png` ist ein einfaches, generiertes 2×2-Atlas für Raketenabgas,
Einschlag, Rauch und Terraintrümmer. Das Sheet ist ein Prototyp-Asset und keine
verbindliche Qualitätsreferenz für die spätere Produktionsgrafik.

Die vollständige Generierungsanweisung steht in `docs/ASSET_GENERATION.md`.
